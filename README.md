# 🖤✨ Happy Birthday Humaira!

A luxurious black & gold interactive birthday website featuring 15 real photos.

## Features
- **Black & gold theme** with pink & white accents — elegant & unique
- Twinkling star particle background on every page
- **Page 1** — Dramatic opening: "Something beautiful was made for you"
- **Page 2** — "HOW DARE YOU" if you click No (with her actual photo shaking 😂)
- **Page 3** — Infinite auto-scrolling polaroid photo wall (3 rows, all 15 photos, tap to zoom!)
- **Page 4** — Birthday tribute with photo grid + gold confetti explosion
- **Page 5** — Final love note with hero photo, shimmer name overlay & floating particles
- All photos are embedded — works offline, no internet needed

## How to use
Just open `index.html` in any browser!

## Deploy to GitHub Pages
1. Create a new GitHub repo
2. Upload this folder's contents
3. Go to Settings → Pages → Source: main branch / root
